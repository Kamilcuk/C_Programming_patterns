# Contributing to C Programming Patterns

#### This project aims to build a collection of unique both basic and advanced programming strategies and patterns implemented in pure C and C standard library only.
#### The goal is create an Educational reference for learners and also to build a collection of creative and experimental programs.

## What to Contribute

- implementation of Data structures
- implementation of Algorithms
- Various Programming Strategies 
- State machines
- Mathematical utilities 
- Simulations
- other creative and interesting C programs
- Documentation

## Contributors are encouraged to submit innovative, unusual, quirky and experimental programs.
Contribute anything you consider cool.

## Philosophy for the project:
The main philosophy for the project is that each program must written in pure C and C standard library.

## Directory Structure
Please place the new files in the appropriate category.
If your contribution does not fit an existing folder, feel free to create a new one.
If necessary, you can create a completely folder provided your programs spans multiple files.
