# C Programming Patterns

## Why does this repository exists ?
This repository was created mostly as a personal recreational project to explore fundamental and advanced C programming strategies and structures.
Hence the repository in its current state is a collection of foundational programs, structures and some complex programs. 
The major philosophy for this project has been to rebuild foundational software in pure C language and its standard library to better understand the language and foundational design strategies used the modern software.

## Why am I open to contribution from others ?
Along the way I figured this repository should not just be limited to a singular person's ideas and imagination. 
So it would be nice to have others contribute soemthing they think is innovative, creative or perhaps deserves to explored.
One can also contribute something to better understand how opensource works.

## Contribution
Anyone is free to contribute their code as long as its written in pure C and only its standard library. 
Or you can help with the documentation or maybe you optimize/expand the existing functionality.
You can fork it, make changes and open a pull request. And after I review it once it will be added to the main branch.

## Note 
Keep in mind, I am just as much a beginner as you are or you were when you started out. So being a little kind and a little help will also help me become a better opensource software maintainer.
